# Predicting Human Chess Blunders

This project investigates the causes of human error in chess and explores whether machine learning models can predict blunders before they occur.

Using large datasets of online chess games and analysis from the Stockfish chess engine, this research examines how factors such as position complexity, branching factor, time pressure, and evaluation volatility influence the probability of human mistakes.

## Research Questions

- Can machine learning models predict human chess blunders?
- What characteristics of a chess position increase the likelihood of human error?
- How do time pressure and position complexity interact to influence mistakes?

## Dataset

Games are sourced from public datasets from Lichess and Chess.com, containing millions of games and move sequences.

## Methods

- Position analysis using the Stockfish chess engine
- Feature extraction from board states
- Machine learning models for error prediction
- Statistical analysis of mistake patterns

## Project Structure

src/ – core Python code  
data/ – game datasets  
notebooks/ – exploratory analysis  
models/ – trained ML models  
results/ – graphs and evaluation metrics

## Goals

The goal of this project is to better understand human decision making under complexity and develop predictive models that estimate the probability of chess blunders.
